Java
----
Fast, secure, reliable programming language.
Portability is one of the key aspects.
	Java apps run similarily on different platforms.
	Write Once, Run Anywhere.
	Achieved using Java bytecode.
JDK (Java Development Kit)
	provides compiler, debugger, libraries, javadocs
	JRE (Java Runtime Environment)
		provides libraries for assisting JVM in executing apps.
		JVM (Java Virtual Machine)
			executes Java applications.
