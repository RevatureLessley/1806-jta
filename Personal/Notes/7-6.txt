HTTPMethods
-----------
GET - get data
POST - insert data
PUT - update data
DELETE - delete data
OPTIONS - return what HTTP methods are allowed
HEAD - return server header information
TRACE - return every destination packets hit roundtrip
PATCH - like POST

Servlet config is an object that describes a servlet. Servlet context is an object that describes all servlets.

OLTP -Online Transaction Processing
     -more normalized (3NF)
     -used for application data

OLAP -Online Analytic Processing
     -more denormalized (1-2NF)
     -used for datawarehouses and datamarts
